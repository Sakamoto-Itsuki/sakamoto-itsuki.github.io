Pannellumを使用した360度画像の埋め込み

手順

・サーバに画像をアップロード

・画像URLまたはファイルパスを取得

・html、js、cssを編集及び360度画像の埋め込み

html

画像は、#panorama=以降にURLで指定します。

      
        <body>
          <iframe
            width="600"
            height="400"
            allowfullscreen
            style="border-style: none"
            src="https://cdn.pannellum.org/2.5/pannellum.htm#panorama=https://sakamoto-itsuki.github.io/image/fy2023-6.jpg"
          ></iframe>
        </body>

API(javascript)

画像は、URLまたはファイルパスで指定します。

ライブラリの読み込みやcssの設定が必要になります。

      
        <head>
          <link
            rel="stylesheet"
            href="https://cdn.jsdelivr.net/npm/pannellum@2.5.6/build/pannellum.css"
          />
          <script
            type="text/javascript"
            src="https://cdn.jsdelivr.net/npm/pannellum@2.5.6/build/pannellum.js"
          ></script>
          <style>
            #panorama {
              width: 600px;
              height: 400px;
            }
          </style>
        </head>
        <body>
          <div id="panorama"></div>
          <script>
            pannellum.viewer("panorama", {
              type: "equirectangular",
              panorama: "image/fy2023-6.jpg",
            });
          </script>
        </body>

補足

html、jsともに自動ロード設定や自動回転などの設定も可能です。

下図は自動ロード、自動回転、タイトル表示、コンパス非表示の設定を追加しています。 

      
        <div id="panorama2"></div>
        <script>
          pannellum.viewer("panorama2", {
            type: "equirectangular",
            panorama: "image/fy2023-6.jpg",
            autoLoad: true,
            autoRotate: -2,
            title: "タイトル",
            compass: false,
          });
        </script>
